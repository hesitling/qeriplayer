# Core Infrastructure Module (core/)

## Overview

The core infrastructure module provides the fundamental services required by all other modules: network communication, database access, file operations, cryptography, and logging.

## Module Composition

```
src/core/
в”ңв”Җв”Җ network/     # HTTP client, WebSocket, network monitoring
в”ңв”Җв”Җ database/    # SQLite wrapper with schema migrations
в”ңв”Җв”Җ filesystem/  # App paths, safe file I/O, file watcher
в”ңв”Җв”Җ crypto/      # AES-256-GCM encryption, secure storage, hashing
в””в”Җв”Җ logger/      # spdlog-based logging with daily rotation
```

## Submodule Documents

- [Network Module](network.md) вҖ” HttpClient (QCoro), WebSocketClient, NetworkManager, NetworkMonitor
- [Database Module](database.md) вҖ” DatabaseManager (sqlite3), parameterized queries, transactions, migrations
- [Filesystem Module](filesystem.md) вҖ” AppPaths, FileUtils (atomic writes), FileWatcher
- [Crypto Module](crypto.md) вҖ” Encryptor/Decryptor (AES-256-GCM), SecureStorage, CryptoUtils (SHA-256)
- [Logger Module](logger.md) вҖ” spdlog wrapper with named loggers, daily file rotation, colored console
